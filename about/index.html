Um Pouco Sobre Mim

Ola! Me chamo Rogerio Medeiros, sou um desenvolvedor web/mobile, nasci no estado do Piauí, e atualmente moro em Niteroi-RJ e trabalho na cidade do Rio de Janeiro.

Trabalho com programção desde 2009, fiquei focado por um bom tempo no back-end, javascript, html, css e há algum tempo venho estudando mobile.

Sobre este site

Este site foi feito utilizando Octopress, um framework open-source para blogs tendo como base Jekyll, e hospedado no Github Pages.

Minha Filosofia

Quando se trata de código e software, eu acredito que as pessoas devem ser tão abertas quando as tecnologias que utilizam.

Principalmente quando se trata de web/mobile, hoje praticamente as tecnologias mais acessiveis do planeta. Com base nisso senti que deveria ajudar os outros a aprender como usá-los.

Com isso, irei tentar postar tudo aquilo que consegui de conhecimento nessa minha pequena trajetoria como desenvolvedor, autodidata.
